# Summary Report: "AI/ML/GenAI on AWS Workshop"

### Event Objectives

- Provide comprehensive overview of AWS AI/ML services and capabilities
- Demonstrate practical implementation of machine learning workflows with Amazon SageMaker
- Introduce Generative AI concepts and Amazon Bedrock foundation models
- Share best practices for building production-ready AI/ML solutions

### Speakers

- **Dr. Alex Nguyen** – Senior AI/ML Specialist Solutions Architect, AWS Vietnam
- **Jennifer Liu** – Machine Learning Engineer, AWS Professional Services
- **Marcus Tan** – Generative AI Technical Lead, ASEAN Region

### Key Highlights

#### Welcome & AI/ML Landscape Overview (8:30 – 9:00 AM)

- **Vietnam AI/ML ecosystem**: Current state, adoption rates, and growth opportunities
- **Industry applications**: Healthcare, finance, e-commerce, and manufacturing use cases
- **Talent development**: AWS AI/ML training programs and certification pathways
- **Success stories**: Local companies leveraging AWS AI/ML services

#### Amazon SageMaker Deep Dive (9:00 – 10:30 AM)

##### Core Platform Features
- **Integrated development environment**: SageMaker Studio for unified ML workflow management
- **Data preparation**: Ground Truth for labeling, Feature Store for feature management
- **Model development**: Built-in algorithms, framework support, and distributed training
- **MLOps capabilities**: Pipelines, Model Monitor, and Clarify for explainability

##### Live Demo Highlights
- **End-to-end workflow**: From data ingestion to model deployment
- **Distributed training**: Scaling across multiple instances for faster results
- **A/B testing**: Canary deployments for production model validation
- **AutoML capabilities**: SageMaker Autopilot for automated model selection

#### Generative AI with Amazon Bedrock (10:45 AM – 12:00 PM)

##### Foundation Models Overview
- **Anthropic Claude**: Advanced reasoning and analysis capabilities
- **Meta Llama 2**: Open-source foundation for customization
- **Amazon Titan**: AWS-developed models for text and embeddings
- **Model selection criteria**: Performance, cost, and use case alignment

##### Advanced Techniques
- **Prompt Engineering**:
  - Zero-shot, one-shot, and few-shot learning strategies
  - Chain-of-Thought reasoning for complex problem solving
  - Temperature and token management for output control

- **Retrieval-Augmented Generation (RAG)**:
  - Architecture patterns for knowledge base integration
  - Vector databases and semantic search implementations
  - Context window optimization strategies

##### Bedrock Capabilities
- **Agents**: Multi-step workflow automation with tool integration
- **Guardrails**: Content filtering and safety controls
- **Fine-tuning**: Custom model adaptation with domain-specific data
- **Batch inference**: Cost-effective processing for large datasets

### Key Takeaways

#### Technical Architecture

- **Unified ML platform**: SageMaker eliminates infrastructure complexity
- **MLOps integration**: Automated pipelines reduce manual deployment effort
- **Scalability**: Elastic resources handle varying workload demands
- **Security**: VPC isolation and encryption at rest and in transit

#### Generative AI Best Practices

- **Model selection**: Choose based on specific use case requirements
- **Cost optimization**: Monitor token usage and implement caching strategies
- **Performance tuning**: Adjust parameters for latency vs. accuracy trade-offs
- **Governance**: Implement proper audit trails and compliance controls

#### Implementation Strategy

- **Start small**: Begin with pilot projects to demonstrate value
- **Data quality**: Focus on high-quality training and prompt data
- **Continuous improvement**: Regular model updates and prompt optimization
- **Team training**: Invest in AI/ML skill development
